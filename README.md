# HireProStaff Terms and Conditions Site

A simple, clean static website displaying the HireProStaff Terms and Conditions.

## Live Site

Once GitHub Pages is enabled, the site will be available at:
https://clashick.github.io/hireprostaff-terms-site/

## Setup Instructions

To enable GitHub Pages:
1. Go to repository Settings
2. Navigate to Pages section
3. Under "Source", select "Deploy from a branch"
4. Select "main" branch and "/ (root)" folder
5. Click Save

The site will be live within a few minutes.

## Local Development

Simply open `index.html` in your browser. No build tools or server required.

## Files

- `index.html` - Main HTML structure
- `styles.css` - Professional styling for legal document display
- `script.js` - Loads and displays the terms text file
- `hireprostaff_terms_plain.txt` - The complete terms and conditions text

## Features

- Clean, professional legal document layout
- Responsive design (mobile-friendly)
- Sticky header with page title
- Easy-to-read typography with good line spacing
- Back to top button for easy navigation
- Print-friendly styles
- No external dependencies (vanilla HTML/CSS/JS)
